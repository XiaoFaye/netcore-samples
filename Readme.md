## How to run

1. Get the dotnet CLI https://github.com/dotnet/cli#installers-and-binaries
2. Clone this repo
3. Navigate to the project folder you prefer
4. Make sure you have read the Readme.md file.
5. run `dotnet restore` in the folder
6. run `dotnet run`
