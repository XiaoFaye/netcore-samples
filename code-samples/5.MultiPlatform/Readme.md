5. MultiPlatform
======================

## How to run

1. Navigate to the project folder
2. run `dotnet restore` in the folder
3. run `dotnet build`
4. run `dotnet publish -r {runtime}`, please replace the runtime to one of below:

runtimes:
- win7-x64
- win7-x86
- win10-x64
- win10-x86
- osx.10.10-x64
- osx.10.11-x64
- ubuntu.14.04-x64
